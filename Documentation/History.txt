Clear Unused Timbres
---------------------
Clears all timbres which have Status OFF or Muted
Can be used over one or more combis or one or more combi banks.

SmartSort Timbres
-----------------
Should be added as button in Timbre window.
But also can be used over one or more combis or one or more combi banks.

Sort by:
- Status
- Mute
- MIDI Channel
- Key Zone (Bottom/Top)
- Vel Zone (Bottom/Top)

MG: x Create Gap between different MIDI channel(s)
KG: x Create Gap between different Key zone(s)

o Create no gaps
o Create gap between different MIDI Channels
o Create double gap between different MIDI Channel(s) and gap between different key zone(s)
x Only sort within 8 timbres

E.g.
Timbre MIDI KeyZone    Result for: MG = Off, KG = Off       MG = Off, KG = On      MG = On, KG = On      MG = On, KG = On
  1     2   C1-C3                    3 2 1                       3 - 2 1              3 1 - 2               3 - 1 - - 2
  2     1   C3-C4            
  3     1   C1-C3
